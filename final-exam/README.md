# 最終総合演習

全9章の学習内容を統合した総合演習。
**何度でも繰り返し取り組むことを前提とし、毎回タイムを計って自己ベストを更新しよう。**

---

## 難易度別一覧

| レベル | 対象 | 目安時間 | 内容 |
|--------|------|---------|------|
| [初級編](beginner/README.md) | 各章 mission-1〜2 クリア済み | 5〜10分/問 | 単一ファイル編集・基本操作の組み合わせ |
| [中級編](intermediate/README.md) | 各章の章末演習クリア済み | 15〜30分/問 | 複数ファイル・マクロ・vimgrep・バッファ操作 |
| [上級編](advanced/README.md) | 全章習得・実務で nvim 使用 | 30〜60分/問 | 実際の OSS コードを題材にした大規模リファクタ |

---

## タイムアタックのやり方

1. タイマーをスタートする
2. `exercise/` ディレクトリを開く（`nvim exercise/src/main.rs`）
3. README の課題をすべて完了させる
4. タイマーを止めて `progress.md` に記録する

```markdown
## 最終総合演習
- [x] 初級編 — 🕐 7:42 (初回) / 🕐 4:15 (自己ベスト)
```

---

## 再挑戦のコツ

- 同じ問題でも「別の方法で解く」を意識する
- 2回目以降は `.` やマクロを積極的に使って手数を減らす
- 前回より短いキーストロークで解けたかを意識する
