## 問題 1 — 複数ファイルの一括リファクタ

`exercise-1/src/` 以下の5ファイルに対して、以下の変更を行え。

### 条件

- `vimgrep` + `:cdo` で複数ファイルを横断する
- マクロと `.` 繰り返しを組み合わせる

### 変更内容

1. `use log::` → `use tracing::` に全ファイルで変更する（`vimgrep` + `:cdo`）
2. `log::info!` → `tracing::info!`、`log::error!` → `tracing::error!` に変更する
3. 各ファイルのモジュールドキュメント（ファイル先頭の `//!`）に `//! nvim-dojo v2` の行を追加する（マクロ）
4. すべてのファイルで trailing whitespace を削除する（`:cfdo %s/\s\+$//g | update`）
5. `cargo check` で変更後の確認をする

## 問題 2 — バッファ・ウィンドウ・タブを駆使した同時編集

`exercise-2/` には `api.rs`・`model.rs`・`db.rs` がある。
3ファイルを同時に開いた状態で以下を完了させよ。

### レイアウト

```
┌────────────┬────────────┐
│  api.rs    │  model.rs  │
├────────────┴────────────┤
│         db.rs           │
└─────────────────────────┘
```

### 変更内容

1. `model.rs` の `User` 構造体に `created_at: chrono::DateTime<chrono::Utc>` フィールドを追加する
2. `db.rs` の `insert_user` 関数のシグネチャを `User` の変更に合わせて修正する
3. `api.rs` のハンドラーで `created_at` フィールドを JSON レスポンスに含める
4. 3ファイルを `:wa` で一括保存する

## 問題 3 — マクロ集中特訓

`exercise-3/src/migrations/` に SQL ファイルが10個ある。
各ファイルに同じ変換を施すマクロを作り、すべてに適用せよ。

**変換内容（各ファイル）**:
- ファイル先頭に `-- migration: {ファイル名}` の行を追加する
