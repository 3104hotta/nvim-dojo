## テーマ C — マクロで10ファイルを一括変換

`exercise-3/src/migrations/` の SQL ファイル10個に対して、3種類の変換を施す。

### 問題 C1 — 1ファイルでマクロを設計・記録（12分）

`001_users.sql` を題材にして、以下の変換を **マクロ `qa` に記録** する:

1. ファイル先頭に `-- migration: 001_users.sql` の行を追加
   - ヒント: `gg O -- migration: <C-R>%<Esc>`（`<C-R>%` で現在ファイル名を挿入）
2. `INTEGER` を `BIGINT` に置換（`:%s/INTEGER/BIGINT/g`）
3. `NOT NULL DEFAULT ''` を `NOT NULL DEFAULT ''::text` に置換

`@a` で `001_users.sql` 自身に再実行しても破綻しないことを確認する（冪等性）。

### 問題 C2 — マクロを残り9ファイルに適用（10分）

- `:args migrations/0[02-9]*.sql migrations/010_*.sql`
- `:argdo normal! @a`
- `:wa`

### 問題 C3 — `-- migration:` ヘッダーが正しいファイル名になっているか確認（10分）

C1 のマクロで `<C-R>%` を使えていればファイル名は自動で入っているはず。
入っていなかった場合、各ファイル1行目を quickfix で確認して修正する。

- `:vimgrep /^-- migration:/ migrations/*.sql`
- `:copen` で全エントリを目視確認
- 間違っているものがあれば `:cdo s/old_name/new_name/g | update`
