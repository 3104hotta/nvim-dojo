# ミッション 3 — Ctrl+v（矩形選択）で複数行を一括編集

**難易度**: ★★★  
**目安時間**: 10分

---

## 課題

`exercise.rs` には同じパターンが複数行に並んでいる。
矩形選択（`Ctrl+v`）を使って一括編集せよ。

1. 連続する5行の `let` 変数宣言の先頭に `//` を追加してコメントアウトする
2. 同じく連続する4行の変数宣言の末尾の `;` を削除する（末尾が揃っていることを利用する）
3. 連続する3行の `"debug"` を `"info"` に一括変更する

## 制約

- 課題1・2 は `Ctrl+v` による矩形選択を使うこと（1行ずつ編集禁止）
- 課題3 は `:%s` または `Ctrl+v` どちらでも可（ただし最小キーストロークで）

## ゴール

`goal.rs` と同一の状態にすること。

---

## ヒント

<details>
<summary>ヒントを見る</summary>

**コメントアウト（課題1）の手順**:
1. コメントアウトしたい先頭行の行頭（`^` で移動）へ
2. `Ctrl+v` → `4j` で5行選択
3. `I` → `// ` → `Esc`（`Esc` 後に全行に反映される）

**末尾削除（課題2）の手順**:
1. 対象行の先頭へ
2. `Ctrl+v` → `3j` → `$` → `h`（`;` を選択）
3. `d` で削除

</details>

---

## 解答例

<details>
<summary>解答を見る</summary>

```
課題1:
  /let x1    → 行頭 (^) へ
  Ctrl+v     → 矩形選択開始
  4j         → 5行選択
  I          → 矩形挿入
  // <Esc>

課題2:
  /let a =   → 対象先頭行へ
  Ctrl+v     → 矩形選択開始
  3j         → 4行選択
  $h         → 各行末尾の ';' を選択
  d          → 削除

課題3:
  :%s/"debug"/"info"/g   → 一括置換
  （または Ctrl+v + r"i"n"f"o で文字単位上書き）
```

</details>

---

## 別解

<details>
<summary>別解を見る（上達したら試してみよう）</summary>

- `:norm I// ` コマンドで選択範囲の全行に操作を適用することもできる
- `:'<,'>s/^/\/\/ /` でビジュアル選択範囲の行頭にコメントを追加
- `gv` で直前の選択範囲を再選択できるため、連続した矩形操作が楽になる

</details>
