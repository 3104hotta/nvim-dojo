# 第2章 モードとインサート

## 概要

Vim のモデルの核心は「モード」にある。
挿入する位置・方向・範囲を意識してモードに入ることで、余計なカーソル移動がなくなる。

---

## コマンド早見表

### ノーマルモード → インサートモード

| コマンド | カーソル位置 | 用途 |
|----------|------------|------|
| `i` | カーソルの左 | その場で挿入 |
| `a` | カーソルの右 | カーソル後に追記 |
| `I` | 行頭（非空白） | 行の先頭に挿入 |
| `A` | 行末 | 行末に追記 |
| `o` | 下に新行を作成 | 次の行に新しく書く |
| `O` | 上に新行を作成 | 前の行に新しく書く |
| `s` | カーソル文字を削除して挿入 | 1文字を置き換えつつ書く |
| `S` | 行全体を削除して挿入 | 行をまるごと書き直す |
| `c{motion}` | motion で選択した範囲を削除して挿入 | 範囲を書き直す |
| `C` | カーソルから行末を削除して挿入 | 行後半を書き直す |
| `R` | 上書きモード | 既存テキストを上書き |

### インサートモード内の操作

| コマンド | 動作 |
|----------|------|
| `Ctrl+h` | バックスペース（1文字削除） |
| `Ctrl+w` | 直前の単語を削除 |
| `Ctrl+u` | カーソルから行頭まで削除 |
| `Ctrl+[` または `Esc` | ノーマルモードに戻る |
| `Ctrl+o` | 1コマンドだけノーマルモードの操作を実行 |
| `Ctrl+r{reg}` | レジスタ `reg` の内容を貼り付け |

### ビジュアルモード

| コマンド | 動作 |
|----------|------|
| `v` | 文字選択 |
| `V` | 行選択 |
| `Ctrl+v` | 矩形選択 |
| `gv` | 直前の選択範囲を再選択 |
| `o` | 選択範囲の反対端にカーソルを移動 |

### モード遷移の鉄則

- **ノーマルモードが「家」**：作業が終わったら必ず戻る
- `Esc` よりも `Ctrl+[` の方が押しやすい場合が多い
- インサートモードで長く留まりすぎない（undo の粒度が粗くなる）
