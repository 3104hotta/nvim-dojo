# 第1章 移動の高速化

## 概要

`hjkl` を卒業し、意図した場所へ最小キーストロークで移動できるようになる。
「どこへ行きたいか」を言語化してからキーを押す習慣が身につくと、操作速度が劇的に変わる。

---

## コマンド早見表

### 文字・単語単位

| コマンド | 動作 |
|----------|------|
| `f{c}` | 行内で文字 `c` へ前方ジャンプ（カーソルを `c` の上に置く） |
| `F{c}` | 行内で文字 `c` へ後方ジャンプ |
| `t{c}` | `c` の1文字手前へ前方ジャンプ |
| `T{c}` | `c` の1文字手前へ後方ジャンプ |
| `;` | 直前の `f/F/t/T` を同方向に繰り返す |
| `,` | 直前の `f/F/t/T` を逆方向に繰り返す |
| `w` | 次の単語の先頭へ |
| `W` | 次の WORD の先頭へ（空白区切り） |
| `b` | 前の単語の先頭へ |
| `B` | 前の WORD の先頭へ |
| `e` | 現在の単語の末尾へ |
| `E` | 現在の WORD の末尾へ |

### 行単位

| コマンド | 動作 |
|----------|------|
| `0` | 行頭（列0）へ |
| `^` | 行頭の非空白文字へ |
| `$` | 行末へ |
| `g_` | 行末の非空白文字へ |

### ファイル全体

| コマンド | 動作 |
|----------|------|
| `gg` | ファイル先頭へ |
| `G` | ファイル末尾へ |
| `{N}G` | N 行目へ（例: `42G` → 42行目） |
| `{N}%` | ファイルの N% の位置へ |

### スクロール・画面

| コマンド | 動作 |
|----------|------|
| `Ctrl+d` | 半画面下スクロール（カーソルも移動） |
| `Ctrl+u` | 半画面上スクロール |
| `Ctrl+f` | 1画面下スクロール |
| `Ctrl+b` | 1画面上スクロール |
| `H` | 画面上部の行へ |
| `M` | 画面中央の行へ |
| `L` | 画面下部の行へ |
| `zz` | カーソル行を画面中央に |
| `zt` | カーソル行を画面上部に |
| `zb` | カーソル行を画面下部に |

### マーク・ジャンプ

| コマンド | 動作 |
|----------|------|
| `%` | 対応する括弧・タグへジャンプ |
| `''` | 直前のジャンプ位置へ戻る |
| `Ctrl+o` | ジャンプリストを1つ前へ |
| `Ctrl+i` | ジャンプリストを1つ後へ |
| `m{a}` | 現在位置にマーク `a` を設定 |
| `` `{a} `` | マーク `a` へジャンプ |
