# 第1章 移動の高速化

## 概要

`hjkl` を卒業し、意図した場所へ最小キーストロークで移動できるようになる。
「どこへ行きたいか」を言語化してからキーを押す習慣が身につくと、操作速度が劇的に変わる。

> 💡 **由来を覚えると忘れない**
> 各コマンドには英単語由来のニーモニックがある（例: `w` = **w**ord）。
> 「動作」だけでなく「由来」も意識すると、似たキーが増えても混同しなくなる。

---

## コマンド早見表

### 文字・単語単位

| コマンド | 由来 | 動作 |
|----------|------|------|
| `f{c}` | **f**ind | 行内で文字 `c` へ前方ジャンプ（カーソルを `c` の上に置く） |
| `F{c}` | **F**ind（後方） | 行内で文字 `c` へ後方ジャンプ |
| `t{c}` | **t**ill | `c` の1文字手前へ前方ジャンプ |
| `T{c}` | **T**ill（後方） | `c` の1文字手前へ後方ジャンプ |
| `;` | — | 直前の `f/F/t/T` を同方向に繰り返す |
| `,` | — | 直前の `f/F/t/T` を逆方向に繰り返す |
| `w` | **w**ord | 次の単語の先頭へ |
| `W` | **W**ORD | 次の WORD の先頭へ（空白区切り） |
| `b` | **b**ack | 前の単語の先頭へ |
| `B` | **B**ack（WORD） | 前の WORD の先頭へ |
| `e` | **e**nd | 現在の単語の末尾へ |
| `E` | **E**nd（WORD） | 現在の WORD の末尾へ |

### 行単位

| コマンド | 由来 | 動作 |
|----------|------|------|
| `0` | 列番号 0 | 行頭（列0）へ |
| `^` | 正規表現の行頭 `^` | 行頭の非空白文字へ |
| `$` | 正規表現の行末 `$` | 行末へ |
| `g_` | **g**o + 末尾 | 行末の非空白文字へ |

### ファイル全体

| コマンド | 由来 | 動作 |
|----------|------|------|
| `gg` | **g**o（先頭、強調で2回） | ファイル先頭へ |
| `G` | **G**o（末尾） | ファイル末尾へ |
| `{N}G` | **G**o + 行番号 | N 行目へ（例: `42G` → 42行目） |
| `{N}%` | パーセント | ファイルの N% の位置へ |

### スクロール・画面

| コマンド | 由来 | 動作 |
|----------|------|------|
| `Ctrl+d` | **d**own | 半画面下スクロール（カーソルも移動） |
| `Ctrl+u` | **u**p | 半画面上スクロール |
| `Ctrl+f` | **f**orward | 1画面下スクロール |
| `Ctrl+b` | **b**ackward | 1画面上スクロール |
| `H` | **H**igh | 画面上部の行へ |
| `M` | **M**iddle | 画面中央の行へ |
| `L` | **L**ow | 画面下部の行へ |
| `zz` | (z = 整列、中央) | カーソル行を画面中央に |
| `zt` | z + **t**op | カーソル行を画面上部に |
| `zb` | z + **b**ottom | カーソル行を画面下部に |

### マーク・ジャンプ

| コマンド | 由来 | 動作 |
|----------|------|------|
| `%` | 対応括弧マッチ | 対応する括弧・タグへジャンプ |
| `''` | 直前位置（マーク `'`） | 直前のジャンプ位置へ戻る |
| `Ctrl+o` | **o**lder（古い位置） | ジャンプリストを1つ前へ |
| `Ctrl+i` | `Tab` と同じキーコード | ジャンプリストを1つ後へ |
| `m{a}` | **m**ark | 現在位置にマーク `a` を設定 |
| `` `{a} `` | バッククォート＝マーク参照 | マーク `a` へジャンプ |
