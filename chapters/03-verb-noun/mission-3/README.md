# ミッション 3 — レジスタを使い分けたヤンク・ペースト

**難易度**: ★★★  
**目安時間**: 10分

---

## 課題

`exercise.rs` は関数が複数ある。レジスタを意識してコピー・移動操作を行え。

1. `fn alpha` の関数本体（`{` 〜 `}` を含む全体）を `fn gamma` の後ろに複製する
2. `fn beta` の戻り値 `-> Result<String, Error>` をヤンクして、`fn delta` の戻り値（現在は `-> ()`）と差し替える
3. `let msg = "hello world";` の文字列だけをヤンクし、その後 `d3w` で別の文字列を削除してから（デフォルトレジスタが上書きされる）ヤンクした文字列を貼り付ける（ヤンクレジスタ `"0` を使う）
4. システムクリップボードに `fn public_api` 関数のシグネチャ（`pub fn public_api(...) -> ...` の行）をコピーする（`"+y` を使う）

## 制約

- 課題3では必ず `"0p` を使うこと（`p` で貼り付けると上書きされたレジスタが使われてしまうことを体験する）
- 課題4では `"+yy` または `"+y$` を使う

## ゴール

`goal.rs` と同一の状態にすること（課題4はクリップボードに入っていれば OK）。

---

## ヒント

<details>
<summary>ヒントを見る</summary>

- `"0p` = ヤンクレジスタ（`"0`）から貼り付け。`d` や `c` で上書きされない
- `"+y` = システムクリップボード（`"+`）へヤンク
- `:reg` でレジスタの内容を確認できる
- 関数全体のヤンク: 関数先頭行で `V` → `%`（対応する `}` まで） → `y`

</details>

---

## 解答例

<details>
<summary>解答を見る</summary>

```
課題1:
  /fn alpha → Vf{%y → /fn gamma → }p

課題2:
  /fn beta → f> → yt<CR> （戻り値部分をヤンク）
  /fn delta → f> → vt{→ p

課題3:
  /hello world → f" → yi" （"0 レジスタに入る）
  /別の文字列 → d3w
  "0p           → ヤンクレジスタから貼り付け

課題4:
  /pub fn public_api → "+yy
```

</details>

---

## 別解

<details>
<summary>別解を見る（上達したら試してみよう）</summary>

- 名前つきレジスタ `"a` 〜 `"z` に保存して使い回す: `"ayy` → `:reg a` で確認 → `"ap`
- `"Ayy` で大文字のレジスタ名を使うと追記できる（上書きでなく）
- `:let @a = @+` でクリップボードの内容をレジスタ a にコピー

</details>
