# 第3章 動詞＋名詞

## 概要

Vim の編集コマンドは **「動詞（operator）+ 名詞（text object / motion）」** の組み合わせで成り立つ。
この文法を身につけると、「ここからここまで」を脳で言語化したものがそのままコマンドになる。

---

## コマンド早見表

### 動詞（Operator）

| コマンド | 由来 | 動作 |
|----------|------|------|
| `d` | **d**elete | 削除（レジスタに保存） |
| `c` | **c**hange | 変更（削除してインサートモードへ） |
| `y` | **y**ank | ヤンク（コピー） |
| `v` | **v**isual | ビジュアル選択 |
| `=` | （整形記号） | インデント整形 |
| `>` / `<` | 矢印 | インデント増加 / 減少 |
| `gU` | **g**o + **U**pper | 大文字化 |
| `gu` | **g**o + **u**pper の小文字版 | 小文字化 |
| `g~` | **g**o + チルダ（トグル） | 大文字・小文字を反転 |

### 名詞：テキストオブジェクト

接頭語 `i` = **i**nner（内側）、`a` = **a**round（外側＝囲い文字を含む）

| コマンド | 由来 | 対象 |
|----------|------|------|
| `iw` / `aw` | inner/around **w**ord | 単語 |
| `iW` / `aW` | inner/around **W**ORD | WORD（空白区切り） |
| `is` / `as` | inner/around **s**entence | 文 |
| `ip` / `ap` | inner/around **p**aragraph | 段落 |
| `i"` / `a"` | クォート文字 | ダブルクォート内（/ 含む） |
| `i'` / `a'` | クォート文字 | シングルクォート内（/ 含む） |
| `` i` `` / `` a` `` | クォート文字 | バッククォート内（/ 含む） |
| `i(` / `a(` | 括弧文字（`ib`/`ab` も可、b = **b**racket） | 丸括弧内 |
| `i[` / `a[` | 角括弧 | 角括弧内 |
| `i{` / `a{` | 波括弧（`iB`/`aB` も可、B = **B**race） | 波括弧内 |
| `i<` / `a<` | 山括弧 | 山括弧内 |
| `it` / `at` | inner/around **t**ag | HTMLタグ内（/ 含む） |

### 組み合わせ例

| コマンド | 意味 |
|----------|------|
| `diw` | カーソルの単語を削除 |
| `ci"` | ダブルクォート内を変更 |
| `ya{` | 波括弧ブロックをヤンク（括弧も含む） |
| `=ip` | 段落のインデントを整形 |
| `gUiw` | カーソルの単語を大文字化 |
| `d2w` | 2単語分削除 |
| `ct,` | `,` の手前まで変更 |
| `yi(` | 丸括弧内をヤンク |

### ペースト

| コマンド | 由来 | 動作 |
|----------|------|------|
| `p` | **p**aste / **p**ut（後） | カーソルの後（下）に貼り付け |
| `P` | **P**ut（前） | カーソルの前（上）に貼り付け |
| `"0p` | レジスタ `0`（ヤンク専用） | ヤンクレジスタから貼り付け |
| `"+p` | `+` レジスタ＝システムクリップボード | システムクリップボードから貼り付け |
| `"+y` | 同上 | システムクリップボードにヤンク |
