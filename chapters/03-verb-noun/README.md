# 第3章 動詞＋名詞

## 概要

Vim の編集コマンドは **「動詞（operator）+ 名詞（text object / motion）」** の組み合わせで成り立つ。
この文法を身につけると、「ここからここまで」を脳で言語化したものがそのままコマンドになる。

---

## コマンド早見表

### 動詞（Operator）

| コマンド | 動作 |
|----------|------|
| `d` | 削除（レジスタに保存） |
| `c` | 変更（削除してインサートモードへ） |
| `y` | ヤンク（コピー） |
| `v` | ビジュアル選択 |
| `=` | インデント整形 |
| `>` / `<` | インデント増加 / 減少 |
| `gU` | 大文字化 |
| `gu` | 小文字化 |
| `g~` | 大文字・小文字を反転 |

### 名詞：テキストオブジェクト

| コマンド | 対象 |
|----------|------|
| `iw` / `aw` | 単語（inner / around ＝ 前後スペース含む） |
| `iW` / `aW` | WORD（空白区切り） |
| `is` / `as` | 文（sentence） |
| `ip` / `ap` | 段落 |
| `i"` / `a"` | ダブルクォート内（/ 含む） |
| `i'` / `a'` | シングルクォート内（/ 含む） |
| `` i` `` / `` a` `` | バッククォート内（/ 含む） |
| `i(` / `a(` | 丸括弧内（`ib` / `ab` でも可） |
| `i[` / `a[` | 角括弧内 |
| `i{` / `a{` | 波括弧内（`iB` / `aB` でも可） |
| `i<` / `a<` | 山括弧内 |
| `it` / `at` | HTMLタグ内（/ 含む） |

### 組み合わせ例

| コマンド | 意味 |
|----------|------|
| `diw` | カーソルの単語を削除 |
| `ci"` | ダブルクォート内を変更 |
| `ya{` | 波括弧ブロックをヤンク（括弧も含む） |
| `=ip` | 段落のインデントを整形 |
| `gUiw` | カーソルの単語を大文字化 |
| `d2w` | 2単語分削除 |
| `ct,` | `,` の手前まで変更 |
| `yi(` | 丸括弧内をヤンク |

### ペースト

| コマンド | 動作 |
|----------|------|
| `p` | カーソルの後（下）に貼り付け |
| `P` | カーソルの前（上）に貼り付け |
| `"0p` | ヤンクレジスタから貼り付け（直前のヤンク） |
| `"+p` | システムクリップボードから貼り付け |
| `"+y` | システムクリップボードにヤンク |
