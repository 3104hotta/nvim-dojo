# 章末総合演習 — operator + text object 連続適用でリファクタ

**難易度**: ★★★  
**目安時間**: 20〜25分

---

## シナリオ

レガシーな Rust コード `exercise.rs` を新しいスタイルにリファクタする。
「動詞＋名詞」の組み合わせを意識して、最小手数で変更を完了させよ。

## 課題

以下の10項目をすべて完了させること:

1. すべての `unwrap()` 呼び出し（5箇所）を `?` に変更する
2. `String::from("...")` パターンを `.to_string()` に変更する（3箇所）
3. `match x { Ok(v) => v, Err(_) => panic!(...) }` ブロックを `x.unwrap_or_else(|_| panic!(...))` に変更する
4. `fn old_name` を `fn new_name` にリネームする（`ciw` を使う）
5. `pub use` で再エクスポートしている型リスト `{TypeA, TypeB, TypeC}` に `TypeD` を追加する（`ci{` を使う）
6. コメントアウトされているコードブロック（5行）のコメント記号 `// ` を矩形選択で一括削除する
7. `derive` マクロの属性 `#[derive(Debug)]` を `#[derive(Debug, Clone, PartialEq)]` に変更する
8. ドキュメントコメント `/// old description` を新しい説明に書き直す
9. `impl Trait for Struct` の `Trait` と `Struct` を入れ替える（レジスタを使う）
10. ファイル全体のインデントを `=G`（gg から G までインデント整形）で整える

## 制約

- 課題1 は `cgn` + `.` で繰り返すこと（5回 `ciw` を繰り返す禁止）
- 課題6 は `Ctrl+v` の矩形選択を使うこと

## ゴール

`goal.rs` と同一の状態にすること。
