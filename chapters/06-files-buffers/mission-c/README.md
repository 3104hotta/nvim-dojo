# ミッション C — ウィンドウサイズ：タイムアタックでレイアウト再現

**難易度**: ★★★  
**目安時間**: 5分以内（目標タイム）

---

## 概要

指定されたレイアウトを、キーボードだけで最短時間で再現するタイムアタック。
何度でも繰り返して自己ベストを更新しよう。

---

## ターゲットレイアウト

```
┌──────────────────────────┬──────────────────────────┐
│                          │         right-top.rs      │
│        left.rs           ├──────────────────────────┤
│                          │       right-bottom.rs     │
└──────────────────────────┴──────────────────────────┘
         （幅50%）                    （幅50%）
   上下分割なし              上下に均等分割（各50%高さ）
```

つまり:
- 左ペイン: `left.rs`（画面の左半分）
- 右上ペイン: `right-top.rs`（画面の右上半分）
- 右下ペイン: `right-bottom.rs`（画面の右下半分）

---

## ルール

1. nvim を起動したときの状態は「`left.rs` を1ウィンドウで開いた状態」
2. タイマーをスタートして、上記のレイアウトを完成させる
3. 完成条件: 3ファイルが正しいペインに開いていること
4. 完成したら `:q` して時間を記録する

## 目標タイム

| レベル | タイム |
|--------|--------|
| 入門   | 60秒以内 |
| 標準   | 30秒以内 |
| 上級   | 15秒以内 |

---

## ヒント

<details>
<summary>ヒントを見る</summary>

- `:vsp right-top.rs` で左右分割
- `Ctrl+w l` で右へ → `:sp right-bottom.rs` で右ペインを上下分割
- `Ctrl+w =` で均等化

</details>

---

## 最小キーストローク解答例

<details>
<summary>解答を見る</summary>

```
キーストローク数: 約45打鍵

:vsp right-top.rs<CR>     （左右分割）
Ctrl+w l                  （右へ移動）
:sp right-bottom.rs<CR>   （上下分割）
Ctrl+w =                  （均等化）
```

**さらに短くする方法**:
```
:vsp right-top.rs | wincmd l | sp right-bottom.rs | wincmd =
```
これで1行で完成（コマンドライン上で `|` でコマンドをつなぐ）。

</details>
