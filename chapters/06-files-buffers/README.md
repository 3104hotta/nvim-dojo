# 第6章 ファイル・バッファ・ウィンドウ

## 概要

複数ファイルを同時に扱う技術を身につける。
netrw でのファイル探索、バッファ操作、分割ウィンドウ、そしてマウスなしでの画面レイアウト構築まで習得する。

---

## コマンド早見表

### netrw（ファイラ）

| コマンド | 動作 |
|----------|------|
| `:Ex` | 現在ファイルのディレクトリを netrw で開く |
| `:Sex` | 水平分割で netrw を開く |
| `:Vex` | 垂直分割で netrw を開く |
| `:Tex` | 新タブで netrw を開く |
| `Enter` | ファイルを開く / ディレクトリに入る |
| `-` | 親ディレクトリへ |
| `%` | 新ファイルを作成 |
| `d` | 新ディレクトリを作成 |
| `D` | ファイル・ディレクトリを削除 |
| `R` | ファイルをリネーム |
| `i` | 表示スタイルを切り替え（thin / long / wide / tree） |

### バッファ

| コマンド | 動作 |
|----------|------|
| `:ls` | バッファ一覧を表示 |
| `:b {N}` | N 番のバッファへ切り替え |
| `:b {name}` | 名前でバッファへ切り替え（Tab 補完可） |
| `:bn` | 次のバッファへ |
| `:bp` | 前のバッファへ |
| `:bd` | 現在のバッファを削除 |
| `:bd {N}` | N 番のバッファを削除 |
| `Ctrl+^` | 直前のバッファに切り替え（交互切り替え） |
| `:e {file}` | ファイルを開く（バッファに追加） |
| `:w` | 保存 |
| `:wa` | 全バッファを保存 |

### ウィンドウ分割

| コマンド | 動作 |
|----------|------|
| `:sp {file}` | 水平分割で開く |
| `:vsp {file}` | 垂直分割で開く |
| `Ctrl+w s` | 現在のバッファを水平分割 |
| `Ctrl+w v` | 現在のバッファを垂直分割 |
| `Ctrl+w h/j/k/l` | 左/下/上/右のウィンドウへ移動 |
| `Ctrl+w w` | 次のウィンドウへ（ローテーション） |
| `Ctrl+w p` | 直前のウィンドウへ |
| `Ctrl+w q` | 現在のウィンドウを閉じる |
| `Ctrl+w o` | 他のウィンドウを全て閉じる（only） |
| `Ctrl+w x` | 隣のウィンドウと入れ替え |
| `Ctrl+w r` | ウィンドウを回転 |

### ウィンドウサイズ変更

| コマンド | 動作 |
|----------|------|
| `Ctrl+w +` | 高さを1行増やす |
| `Ctrl+w -` | 高さを1行減らす |
| `Ctrl+w >` | 幅を1列増やす |
| `Ctrl+w <` | 幅を1列減らす |
| `{N}Ctrl+w +` | 高さを N 行増やす |
| `{N}Ctrl+w >` | 幅を N 列増やす |
| `Ctrl+w =` | 全ウィンドウを均等化 |
| `Ctrl+w _` | 高さを最大化 |
| `Ctrl+w \|` | 幅を最大化 |
| `:resize {N}` | 高さを N 行に指定 |
| `:vertical resize {N}` | 幅を N 列に指定 |

### タブ

| コマンド | 動作 |
|----------|------|
| `:tabnew {file}` | 新タブでファイルを開く |
| `gt` | 次のタブへ |
| `gT` | 前のタブへ |
| `{N}gt` | N 番目のタブへ |
| `:tabclose` | 現在のタブを閉じる |
| `:tabonly` | 他のタブを全て閉じる |
| `:tabs` | タブ一覧を表示 |
