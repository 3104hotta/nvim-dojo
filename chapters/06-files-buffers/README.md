# 第6章 ファイル・バッファ・ウィンドウ

## 概要

複数ファイルを同時に扱う技術を身につける。
netrw でのファイル探索、バッファ操作、分割ウィンドウ、そしてマウスなしでの画面レイアウト構築まで習得する。

---

## コマンド早見表

### netrw（ファイラ）

| コマンド | 由来 | 動作 |
|----------|------|------|
| `:Ex` | **Ex**plore | 現在ファイルのディレクトリを netrw で開く |
| `:Sex` | **S**plit + **Ex**plore | 水平分割で netrw を開く |
| `:Vex` | **V**ertical + **Ex**plore | 垂直分割で netrw を開く |
| `:Tex` | **T**ab + **Ex**plore | 新タブで netrw を開く |
| `Enter` | 開く | ファイルを開く / ディレクトリに入る |
| `-` | 親ディレクトリ | 親ディレクトリへ |
| `%` | （新規ファイル） | 新ファイルを作成 |
| `d` | **d**irectory | 新ディレクトリを作成 |
| `D` | **D**elete | ファイル・ディレクトリを削除 |
| `R` | **R**ename | ファイルをリネーム |
| `i` | (内部の "i" コマンド) | 表示スタイル切り替え（thin / long / wide / tree） |

### バッファ

| コマンド | 由来 | 動作 |
|----------|------|------|
| `:ls` | **l**i**s**t（バッファ一覧）| バッファ一覧を表示 |
| `:b {N}` | **b**uffer | N 番のバッファへ切り替え |
| `:b {name}` | **b**uffer | 名前でバッファへ切り替え（Tab 補完可） |
| `:bn` | **b**uffer **n**ext | 次のバッファへ |
| `:bp` | **b**uffer **p**revious | 前のバッファへ |
| `:bd` | **b**uffer **d**elete | 現在のバッファを削除 |
| `:bd {N}` | 同上 | N 番のバッファを削除 |
| `Ctrl+^` | **a**lternate file（直前バッファ）| 直前のバッファに切り替え |
| `:e {file}` | **e**dit | ファイルを開く（バッファに追加） |
| `:w` | **w**rite | 保存 |
| `:wa` | **w**rite **a**ll | 全バッファを保存 |

### ウィンドウ分割

| コマンド | 由来 | 動作 |
|----------|------|------|
| `:sp {file}` | **sp**lit | 水平分割で開く |
| `:vsp {file}` | **v**ertical **sp**lit | 垂直分割で開く |
| `Ctrl+w s` | **w**indow + **s**plit | 現在のバッファを水平分割 |
| `Ctrl+w v` | **w**indow + **v**ertical | 現在のバッファを垂直分割 |
| `Ctrl+w h/j/k/l` | window + 方向キー | 左/下/上/右のウィンドウへ移動 |
| `Ctrl+w w` | **w**indow + next **w**indow | 次のウィンドウへ（ローテーション） |
| `Ctrl+w p` | **p**revious | 直前のウィンドウへ |
| `Ctrl+w q` | **q**uit | 現在のウィンドウを閉じる |
| `Ctrl+w o` | **o**nly | 他のウィンドウを全て閉じる |
| `Ctrl+w x` | e**x**change | 隣のウィンドウと入れ替え |
| `Ctrl+w r` | **r**otate | ウィンドウを回転 |

### ウィンドウサイズ変更

| コマンド | 由来 | 動作 |
|----------|------|------|
| `Ctrl+w +` | + で増やす | 高さを1行増やす |
| `Ctrl+w -` | - で減らす | 高さを1行減らす |
| `Ctrl+w >` | 右矢印 | 幅を1列増やす |
| `Ctrl+w <` | 左矢印 | 幅を1列減らす |
| `{N}Ctrl+w +` | 数値 + 増 | 高さを N 行増やす |
| `{N}Ctrl+w >` | 数値 + 増 | 幅を N 列増やす |
| `Ctrl+w =` | イコール（均等）| 全ウィンドウを均等化 |
| `Ctrl+w _` | アンダースコア（横一杯）| 高さを最大化 |
| `Ctrl+w \|` | パイプ（縦一杯）| 幅を最大化 |
| `:resize {N}` | **resize** | 高さを N 行に指定 |
| `:vertical resize {N}` | vertical + resize | 幅を N 列に指定 |

### タブ

| コマンド | 由来 | 動作 |
|----------|------|------|
| `:tabnew {file}` | tab + new | 新タブでファイルを開く |
| `gt` | **g**o + **t**ab | 次のタブへ |
| `gT` | **g**o + **T**ab（逆方向） | 前のタブへ |
| `{N}gt` | 数値 + go tab | N 番目のタブへ |
| `:tabclose` | tab + close | 現在のタブを閉じる |
| `:tabonly` | tab + **only**（残す） | 他のタブを全て閉じる |
| `:tabs` | tabs（一覧） | タブ一覧を表示 |
