# ミッション 2 — `:%s` で正規表現置換

**難易度**: ★★☆  
**目安時間**: 10分

---

## 課題

`exercise.rs` を正規表現置換で変換せよ。

1. すべての `println!` を `eprintln!` に変更する（単純置換）
2. `snake_case_name` のパターンをすべてキャメルケース `snakeCaseName` に変換する（正規表現 + キャプチャグループ）
3. 行末の余分な空白（trailing whitespace）をすべて削除する
4. `// コメント` の `//` と文字の間にスペースがない箇所（`//コメント`）に空白を挿入する
5. `v0.1.0` のようなバージョン文字列を `v0.2.0` に変更する（数値の一部だけ変更）

## 制約

- 課題2 はキャプチャグループ `\(pattern\)` と `\1` を使うこと
- 各 `:%s` コマンドを実行前に `:noh` して結果を確認すること

## ゴール

`goal.rs` と同一の状態にすること。

---

## ヒント

<details>
<summary>ヒントを見る</summary>

- 課題3（trailing whitespace）: `:%s/\s\+$//g`
- 課題4（`//` 後のスペース）: `:%s|//\([^ ]\)|// \1|g`
- 課題2（スネーク→キャメル）は複数ステップに分けると楽:
  - `:%s/_\([a-z]\)/\u\1/g` で `_x` を `X` に変換（`\u` = 次の文字を大文字化）
- 課題5: `:%s/v0\.1\.0/v0.2.0/g`（`.` はエスケープ必要）

</details>

---

## 解答例

<details>
<summary>解答を見る</summary>

```
1. :%s/println!/eprintln!/g

2. :%s/_\([a-z]\)/\u\1/g

3. :%s/\s\+$//g

4. :%s|//\([^ /]\)|// \1|g

5. :%s/v0\.1\.\zs0/2/g
   （\zs でマッチ開始位置を指定して最小限の置換）
```

</details>

---

## 別解

<details>
<summary>別解を見る（上達したら試してみよう）</summary>

- `\v` (very magic) モードを使うと正規表現がより直感的に書ける:
  `:%s/\v_([a-z])/\u\1/g`
- `:%s/old/new/gc` の `c` フラグで確認しながら置換（`y`=yes, `n`=no, `a`=all, `q`=quit）
- `:'<,'>s/pattern/replace/g` でビジュアル選択範囲だけ置換

</details>
